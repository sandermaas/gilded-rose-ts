# Gilded Rose

This is the Gilded Rose kata in TypeScript.

## Getting started

Install dependencies

```sh
pnpm install
```

## Run the unit tests from the Command-Line

```sh
pnpm test
```

To run the Text test

```sh
pnpm test:text
```

To run all tests in watch mode

```sh
pnpm test:watch
```

## Format the code with Prettier

```sh
pnpm format
```
